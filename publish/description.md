## Full description
*(max 4000 characters — the main store listing body)*

```
UTX Wallet is a self-custodial Bitcoin wallet built for users who take their sovereignty seriously. Your keys never leave your device, your transactions stay private, and you choose which node validates your data.

─── YOUR KEYS, YOUR BITCOIN ───

UTX Wallet generates and stores your seed phrase exclusively on your device using encrypted storage. Private keys are derived from your seed only at the moment of signing and are never persisted anywhere — not in memory, not on disk.

Create a new wallet from a freshly generated 12-word BIP39 seed, or import any existing wallet with a 12 or 24-word seed phrase. BIP39 passphrase (25th word) is fully supported for an additional layer of security.

─── PERSONAL NODE — TRUE PRIVACY ───

When you use a public Bitcoin API, your addresses and transaction history are sent to external servers. UTX Wallet lets you connect to your own self-hosted Mempool.space instance so that no third party ever sees your wallet data.

• Configure multiple personal nodes with custom priority ordering
• The app tries nodes in priority order — automatic failover if one is unreachable
• Public API fallback is OFF by default — your traffic stays on your own infrastructure
• Built-in node setup guide: Docker, Umbrel, RaspiBlitz, Start9

─── FULL UTXO CONTROL ───

UTX Wallet gives you complete visibility and control over your unspent outputs:

• View every UTXO with its value, address, and confirmation status
• Freeze individual UTXOs to exclude them from coin selection
• Address-grouped coin selection reduces traceability of your funds
• Replace-By-Fee (RBF) to accelerate stuck transactions

─── SEND & RECEIVE ───

• Native SegWit BIP84 addresses (bc1q)
• Fee presets: fast, half-hour, hour, economy — all from real-time blockchain data
• Full transaction preview before signing — see exact fee and recipient amount
• QR code scanning for addresses and wallet import
• Receive screen with shareable QR for any account

─── MULTI-WALLET, MULTI-ACCOUNT ───

Run multiple wallets on the same device. Each wallet supports multiple BIP44 accounts with automatic address discovery — the app scans the blockchain to find your existing transaction history on import.

─── SECURITY ───

• PIN code + biometric (fingerprint / face) protection
• Auto-lock with configurable timeout
• Hide balance across all screens
• Screen capture prevention on sensitive screens
• Watch-only wallet support — import an xpub or output descriptor to monitor a cold wallet without exposing keys

─── OPEN SOURCE ───

UTX Wallet is free, open source, and contains no ads, no analytics, and no telemetry. Your data belongs to you.

Source code: github.com/misesdev/utx-wallet

─── LANGUAGES ───

English · Português (Brasil)
```

- Does **not** send analytics or telemetry
- Does **not** use advertising SDKs
- Stores wallet data **exclusively on the device** using encrypted storage
- Network requests go only to the user-configured node or to mempool.space (public fallback, disabled by default)
